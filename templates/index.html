Welcome to FHIR Data Generator

This tool allows you to configure and generate FHIR-compliant test data for your EHR sandbox. You can create specific test patients with predefined conditions, medications, and other clinical data.

Configure Patients

Features

  • Configure specific test patients with custom demographics
  • Define pre-existing conditions and comorbidities
  • Specify medications and prescriptions
  • Add allergies and intolerances
  • Control the volume of generated resources (appointments, encounters, observations, etc.)
  • Generate data to JSON file or send directly to FHIR server
